# Frontend

React + TypeScript frontend powered by Vite.

## Scripts

### `npm start`

Starts the Vite development server at [http://localhost:3000](http://localhost:3000).

### `npm test`

Runs the Vitest test suite once.

### `npm run build`

Type-checks the project and builds production assets into the `build` folder.

### `npm run preview`

Serves the production build locally on port 3000.

### `npm run lint:css`

Runs Stylelint for CSS files under `src`.

## Environment

Use `VITE_API_URL` to override the API base URL. In development, `/api` requests are proxied to `http://localhost:8080` by Vite.
